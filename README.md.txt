Первое задание
Краткое описание среды разработки:
1. Язык программирования:
Python 3.10+ для серверной части (подходит для создания API и обработки данных).
JavaScript (Node.js 16.x) для альтернативного варианта серверной части или для фронтенда.

2. Фреймворки и библиотеки:
Express.js (Node.js) — минималистичный фреймворк для разработки серверных приложений.
pyODBC(Python) — для работы с SQL Server.

3. База данных:
SQLite — альтернатива для быстрого старта с базой данных.

4. Инструменты разработки:
Visual Studio Code — редактор кода с расширениями для Python и JavaScript.
Postman — для тестирования API.
Git — для управления версиями.

5. Операционная система:
Приложение разрабатывалось и тестировалось на Windows 10/11 и Ubuntu 22.04.


Второе задание
1)Приложение "Готовь из того, что есть" помогает пользователю подбирать рецепты на основе доступных ингредиентов. 
Программа поддерживает:

Сохранение данных пользователя (JSON файл).
Историю поиска рецептов.
Управление избранными рецептами.
Мониторинг системных ресурсов.
Установка

Клонируйте репозиторий:
git clone <URL>
cd <project_directory>

Установили зависимости:

pip install -r requirements.txt
Запустили сервер Flask:
python app.py
Откроли HTML-файл в браузере для взаимодействия с интерфейсом.


CRUD для файлов: чтение, добавление, обновление и удаление данных.
Работа с JSON: сохранение и загрузка состояния пользователя.
Управление системными процессами: мониторинг ресурсов (CPU, память) и запись данных в лог-файл.
Динамический интерфейс: взаимодействие с сервером через JavaScript для получения рецептов.
